goim
==============
goim is a im server writen by golang.

## Features
 * light weight
 * high performance
 * pure golang implementation
 * heartbeat
 * auth (if a subscriber not auth then can't connect to comet node)
 * multiple protocol (websocket, tcp, http longpolling)
 * stat
